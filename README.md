# #30DaysOfPython 🐍

At [**Indian Data Club**](http://indiandataclub.com), we’re on a mission to hook up the next gen with **AI & Data skills** to crush it in the tech game. Our **30-Day Python Challenge** is your ticket to going from newbie to Python pro, all while building and challenging yourself every day. 

Learn, flex, and share your progress with the squad! 🌟

## **📅 Challenge Breakdown**

- **Duration**: 30 days
- **Focus**: Core Python (syntax, data structures, OOP, advanced features, and practical applications)
- **Structure**: Daily topics paired with problem statements
- **Goal**: Equip you with skills to create web tools, automation scripts, and build a foundation for DA, DS and more! 💻
